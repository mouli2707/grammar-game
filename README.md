# Grammar Island Adventure

Interactive educational game for learning parts of speech through island-hopping challenges.

## Features
- 5 themed islands (Noun, Verb, Adjective, Adverb, Preposition)
- 75 multiple-choice questions
- Progressive difficulty levels
- Sound effects and animations
- Kid-friendly Nintendo-style design

## How to Run
```bash
npm install
npm run dev
```

The game will be available at http://localhost:5000
